# Weather-App
A modern React Weather App providing real-time weather updates with a clean UI. Users can search any city to view temperature, humidity, wind speed, and conditions. It includes API integration, error handling, loading states, and a fully responsive design for all devices.

##  Features
- Real-time weather data
- City search functionality
- Error handling
- Loading state
- Responsive design

##  Technologies Used
- React JS
- Axios
- OpenWeather API
- Tailwind Css

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file and add your API key
4. Run `npm start`
